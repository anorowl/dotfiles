# Dotfiles
My linux environment dotfiles

## Prerequisites

* `rxvt-unicode`
* [`picom-tryone`](https://github.com/tryone144/picom). Location :
`~/.config/picom/picom.conf`
* `polybar`. Location : `~/.config/polybar/{config,launch.sh}`
* `vim`
* `pywal` 
* `mpd`
* `rofi`. Location : `~/.config/rofi/config`
